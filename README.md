# Run Compose
This repository contains an example of a basic chatbot built using docker compose having a web container and an AI model. The project can be deployed onto Cloud Run using `gcloud run compose up` or the Cloud Run button.

[![Run on
Cloud Run](https://deploy.cloud.run/button.svg)](https://deploy.cloud.run/?git_repo=https://github.com/harish-sharma-94/run-compose-demo.git)
